離島機場（七美、望安、蘭嶼、綠島）天氣起降小資訊

  1. 構成飛航天氣的基本因素包括(起飛與降落機場)「風向、風速(側風、尾風)」、「能見度(下雨、霾、霧等因素)」、「雲幕高」及「乾/濕跑道」等。
  2. 離島機場航班航路氣象必須遵守臺北飛航情報區飛航指南(AIP)航路1.2目視飛航規則(VFR)，同時必須依據飛航服務總臺航空氣象單位提供天氣資料(METAR)， 逐班依METAR天氣資料狀況決定：：
    • 【離島航線】: 短航線、目視飛航規則(VFR)、有條件之助/導航機場設施。
    • 【METAR】天氣資料 : 整點發布、特別發布。
    • 【目視機場】: 許可飛航類別為目視飛航規則(VFR)，VMC (目視天 氣)是指 能見度 5000 公尺 及 雲幕高 1500 呎 之天氣狀況，及機型原 廠規定之乾/濕跑道風速(側風、尾風)限制，正側風乾跑道18浬、正 側風濕跑道15浬、不定向風超過起降10浬，及其他各不同度角有 其不同的風速限制。
    • 【航空氣象服務網】- ( 網頁連結 )
    • 【Windy 】- ( 網頁連結 )
  3. 【 什麼是造成航班延誤的『天氣因素』? 】(資料來源: 航空氣象服務網) 從旅客的角度來看，天氣不好就是機場出現大風大雨或大霧，飛機就可能無法起降，航班就要延誤，其實這種認知並不夠精確，因而容易造成誤解，尤其是出現有的航班能走，有的又走不了的情況下。實際上，因『天氣因素』造成航班延誤有很多種情況，如 出發地機場天氣狀況不適合飛機起飛 ，或 航路上的天氣狀況不宜飛機穿越 ，或 目的地機場天氣狀況不適合飛機降落 等，而造成這些情況的天氣因素包括：低能見度、低雲幕高、風切變、晴空亂流、飛機積冰、強降水、強風、雷雨、颱風等。
  4. 【 天氣狀況良好，能見度佳，為什麼還是因天氣因素延誤? 】(資料來源: 航空氣象服務網) 天氣良好，無大雨無大風，航班卻仍然延誤，常使旅客無法理解，其實機場所在地天氣狀況良好，不代表該機場適合航機起降，因為仍有其他原因可能造成航班延誤。例如航機在降落前到達一定高度時，為確保飛行安全，駕駛員必須完全看得見跑道及地面狀況，此時如果駕駛員無法看見跑道，航機將無法降落，而覆蓋在機場起飛降落航道附近的低雲、雷雨區就可能造成此種情況。 還有航機在航路飛行時，倘若不巧碰到航路前面有雷雨的天氣狀況發生，基於安全考量，駕駛員通常不會直接穿過該雷雨區域，而是採取繞過或飛越的方式通過，這就會增加航機之航行時間導致航班延誤。另外，還有靠山邊的機場也常因風切亂流情況，導致飛機降落困難，不是重飛就是返航，這樣航班就會延誤，如蘭嶼機場 。
