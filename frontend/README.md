# AerotwineX 前端系統

這是 AerotwineX 航班查詢系統的前端部分，使用 Vue.js 3 開發。

## 功能特色

- 航班搜尋：根據出發地、目的地、日期和航空公司進行搜尋
- 直覺式界面：黑白極簡風格設計
- 響應式佈局：支援各種尺寸的裝置
- 模擬數據：現階段使用模擬數據展示功能

## 啟動專案

### 安裝依賴
```
npm install
```

### 啟動開發伺服器
```
npm run serve
```

### 編譯生產版本
```
npm run build
```

## 專案結構

- `src/components/` - Vue 組件
- `src/assets/` - 靜態資源
- `public/` - 公共文件

## 技術棧

- Vue.js 3
- Axios (HTTP 請求)
- CSS (使用 scoped 樣式)
- Noto Sans TC 字體
